Overview ▹

Overview ▾

Package salesinvoice implements a simple package for handling all graphql operations related to Microsoft Navision SalesInvoice page.

Package has a type "SalesInvoice" where all the fields related to SalesInvoice are defined. 

Example:
	'''
	type SalesInvoice struct {
		No                       string                `json:"No"`
		SellToCustomerNo         string                `json:"Sell_to_Customer_No"`
		SellToContactNo          string                `json:"Sell_to_Contact_No"`
		...
	}
	'''

GraphQl Object Type along with its fields, arguments and attributes are generated from the SalesInvoice type when "CreateType" method is invoked.
