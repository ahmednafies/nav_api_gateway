Package SalesOrder implements a simple package for handling all graphql operations related to Microsoft Navision SalesOrder page.

Package has a type "SalesOrder" where all the fields related to SalesOrder are defined. 

Example:
	'''
	type SalesOrder struct {
		No                       string                `json:"No"`
		SellToCustomerNo         string                `json:"Sell_to_Customer_No"`
		SellToContactNo          string                `json:"Sell_to_Contact_No"`
		...
	}
	'''

GraphQl Object Type along with its fields, arguments and attributes are generated from the SalesOrder type when "CreateType" method is invoked.
