Overview ▹

Overview ▾

Package salesline implements a simple package for handling all graphql operations related to Microsoft Navision SalesLine page.

Package has a type "SalesLine" where all the fields related to SalesLine are defined. 

Example
	'''
	type SalesLine struct {
		No                  string  `json:"No"`
		DocumentNo          string  `json:"Document_No" required:"true"`
		DocumentType        string  `json:"Document_Type" required:"true"`
		LineNo              int     `json:"Line_No" required:"true"
		...
	}
	'''

GraphQl Object Type along with its fields, arguments and attributes are generated from the SalesLine type when "CreateType" method is invoked.
