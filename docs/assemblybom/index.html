Overview ▹

Overview ▾

Package assemblybom implements a simple package for handling all graphql operations related to Microsoft Navision Assembly_Bom page.

Package has a type "AssemblyBom" where all the fields related to Assembly_Bom are defined. 

Example:

	'''
	type AssemblyBom struct {
		No                string  `json:"No"`
		ParentItemNo      string  `json:"Parent_Item_No"`
		Type              string  `json:"Type"`
		...
	}
	'''

GraphQl Object Type along with its fields, arguments and attributes are generated from the AssemblyBom type when "CreateType" method is invoked.
